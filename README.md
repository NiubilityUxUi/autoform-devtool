# vue-devtools

### 注意事项

> 本项目暂且只对autoform提供开发工具。
> 本项目开发中,暂时只能本地开发模式运行，陆续会实现打包，上传到扩展中心。

### Installation

 1. Install 

	~~~~
	$ npm install 
	~~~~

	Or, for Yarn:

	~~~~
	$ yarn install
	~~~~

 2. Run in Chrome

	~~~~
	$ npm run dev
	~~~~

	When using Yarn, just replace `npm` with `yarn`.

### Usage

 1. chrome 开启 开发者模式 

 2. “加载已解压程序” 选择目录 到：

	~~~~
	./shells/
	~~~~
 3. 打开含有autoform的页面，打开控制台到autoform 刷新页面。
